# MODELS

Folder ini untuk layer **models**.

- **domain** → berisi entity, interface, dan aturan bisnis murni
- **application** → berisi use case, service koordinasi domain
- **infrastructure** → berisi API call, repository, integrasi eksternal
- **presentation** → berisi komponen Angular, routing, dan UI

> Bagian ini adalah bagian dari arsitektur **Clean Architecture** untuk fitur ini.
